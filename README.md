# Million Items Selector

Тестовое задание для компании «Цифровые решения» на позицию Fullstack разработчика.

## Что это

Веб-приложение для выбора и сортировки элементов из списка в 1 000 000 записей.

## Функционал

- **Два окна:** левое (все элементы) и правое (выбранные)
- **Инфинити-скролл** по 20 элементов в каждом окне
- **Фильтрация по ID** в реальном времени
- **Drag & Drop** для сортировки выбранных элементов
- **Сохранение состояния** на сервере между обновлениями страницы
- **Очереди с дедупликацией** и батчингом (1 сек / 10 сек)
- **Добавление новых элементов** с произвольным ID

## Технологии

- **Бэкенд:** Node.js, Express.js
- **Фронтенд:** Чистый JavaScript, HTML5, CSS3
- **Хранение:** In-memory (без БД)
- **Деплой:** Render.com

## Запуск локально

```bash
npm install
npm start
```

Открыть: `http://localhost:3000`

## Демо

Приложение развёрнуто на Render.com:  
https://million-items-selector-xt3j.onrender.com

(Бесплатный тир Render может «засыпать» после 15 минут бездействия. Если страница не загрузилась с первого раза — обновите её.)
